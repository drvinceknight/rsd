Welcome to the instructor documentation
=======================================

This is the instructor documentation for a 2 day workshop for Mathematics
students at Cardiff University's School of Mathematics.

The workshop is carried out over 2 days with the following structure:

- Day 1
  - Motivation of the course;
  - Use of command line and text editors;
  - Basic Python;
  - Programming best practices.
  - LaTeX best practice
- Day 2: morning
  - Version control
  - Group based activity
